App requirements:


The app requires Node.js to be installed on your machine (https://nodejs.org/en/download/).


To build the app:


Extract the contents of the archive.
Open a terminal window in the root folder of the app (word-counter).
Run the command "npm i".
Wait for the installation to finish.


To run the app:


Make sure that the terminal is open in the root folder.
Run the command "npm start <filePath>",
where <filePath> is the location of the file that the app should read.
The result is printed in the terminal console.


To run the tests:


Make sure that the terminal is open in the root folder.
Run the command "npm test".
The results are printed in the terminal console.


A description of the app is located inside the root folder in about.txt.
